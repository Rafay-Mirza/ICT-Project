# 🎬 MovieVerse – ICT Semester Project

MovieVerse is a multi-page movie website created for the ICT Semester Project.  
It includes pages such as Home, Movies, Movie Details, Sign Up, Sign In, and About Us.  
The purpose of this project is to learn website structure, HTML, CSS, GitHub collaboration,  
and deployment using GitHub Pages.

## 📌 Website Pages
1. **Home Page:** Main introduction, navigation, banner  
2. **Movies Page:** A list of movies with posters  
3. **Movie Details Page:** Detailed information for each movie  
4. **Sign Up Page:** User registration form  
5. **Sign In Page:** Login form  
6. **About Us Page:** Website or team information
